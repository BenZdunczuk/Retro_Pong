\doxysubsection{Detailed Description}
Klasa \doxylink{class_main_window}{Main\+Window} reprezentuje główne okno aplikacji Pong. 

Zawiera interfejs graficzny oparty na QMain\+Window, który integruje komponent do gry Pong oraz przycisk sterujący pauzowaniem gry.
